ОПИСАНИЕ
~~~~~~~~
Менеджер виджетов (блоков) позволяет манипулировать виджетами на страницах 
сайта, созданного на базе LiveStreet/ACE

Возможности:
  1) Добавляет виджеты (блоки), используя параметры, заданные в файле 
     конфигурации плагина
  2) Позволяет определять, на каких страницах выводить и/или на каких 
     страницах не надо выводить те или иные виджеты (блоки)
  3) Подгружает, при необходимости, конфигурационные файлы виджетов (блоков),
     лежащие в папке config/widgets/имя_блока (подгрузка осуществляется 
	 только для тех блоков, которые должны выводиться на странице)

Преимущества:
  1) Можно манипулировать отображением виджетов (блоков), редактируя всего 
     один файл конфигурации
  2) Можно создавать и использовать свои виджеты (блоки), которые требуют 
     подгрузки конфигурационных файлов
  2) Можно задавать условия отображения виджетов (блоков)

	В комплекте с модулем идет виджет (блок) BlockUsersOnline, показывающий, 
	какие зарегистрированные пользователи сейчас находятся на сайте.

ИЗМЕНЕНИЯ В НАЗВАНИИ ПАРАМЕТРОВ В ВЕРСИИ 2.x
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
В версии 2.х изменены названия некоторых параметров, которые задаются в
конфигурационном файле. Ниже даны старые названия и новые. Учтите это, если
Вы выполняете апгрейд с версии 1.х
'clear_blocks' -> 'clear'
'blocks' -> 'widgets'
	'block' -> 'name'
	'position' -> 'group'
